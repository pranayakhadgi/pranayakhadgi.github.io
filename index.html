Projects

Truman in the Virtual - Campus Cube

Parents and prospective students need a way to explore campus remotely, especially those unable to visit in person. Traditional static images and videos lack interactivity and spatial awareness.

Developed a 3D virtual campus tour web application using skybox integration, AI voice narration, and map API integration. Implemented inbuilt annotation system for interactive exploration of campus departments and outdoor areas. Built as a research project under Dr. Kaafi Rahman at Truman State University.

Created a functional prototype demonstrating Software as a Service potential for virtual campus tours. The system enables remote exploration of campus facilities and serves as a foundation for expanding to interior touring capabilities.

Memory Garden

Users need a way to track daily mood and emotional patterns over time. Traditional journaling lacks visual feedback and engagement mechanisms.

Built a digital journaling platform using vanilla HTML, CSS, and JavaScript integrated with Generative AI API. Implemented mood tracking system where each journal entry plants a flower representing the user's emotional state, gradually building a visual garden. Stored entries in MongoDB database with SQL query capabilities for data analysis.

Completed as a time-constrained hackathon project. The platform provides users with both written reflection and visual representation of emotional patterns over time, demonstrating proficiency in MongoDB and database management.

Safe Walk Plus

Users need routing that considers both safety and efficiency, especially when traveling at different times of day when risk factors vary.

Developed a machine learning-based routing system that calculates risk scores through ML training. Implemented graph data structure using Dijkstra and A* algorithms to find optimal routes balancing safety and speed based on time of day. Applied graph algorithms for pathfinding and route optimization.

The system provides users with route recommendations that prioritize safety while maintaining reasonable travel time, demonstrating practical application of graph algorithms, Dijkstra, A*, and machine learning in navigation systems.
